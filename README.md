# quicktick-tty ( unstable, abandoned )

A tty frontend for my todo api [Api](https://github.com/osmak1234/quicktick-api)

## Run locally

```
$ git clone https://github.com/osmak1234/quicktick-tty.git

$ cd quicktick-tty

$ cargo build --release

$ ./target/release/quicktick-tty
```
